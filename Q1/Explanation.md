# Question 1 - Linux Environment Verification

## Command: whoami

Explanation:
The whoami command displays the username of the currently logged-in user. The output confirmed that the current user account is Yash.

---

## Command: groups

Explanation:
The groups command shows all groups associated with the user account. This helps verify user permissions and access rights.

---

## Command: echo $SHELL

Explanation:
This command displays the default shell currently being used. The output showed that the Bash shell is being used.

---

## Command: pwd

Explanation:
The pwd command prints the present working directory. It shows the location from where commands are being executed.

---

## Command: ls -la

Explanation:
The ls -la command lists all files and directories, including hidden files, along with detailed information such as permissions and ownership.

---

## Command: ping -c 4 google.com

Explanation:
The ping command checks network connectivity by sending packets to a remote server. Successful replies indicate that the internet connection is working correctly.
